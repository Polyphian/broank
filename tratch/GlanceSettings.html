Glance

Settings

Learn what options are available for Glance, and what they can do for you

Username
This is the name you'd like Glance to greet you as.
Wallpaper
This is the source for your wallpaper. If a theme doesn't include a wallpaper, you can set your own source.

"Auto" = Featured wallpapers

"Custom" = Allows you to choose your own album from your device

You can also type in any subject you'd like to search Unsplash for. You can use things like mountains, bokeh, cars, nature, ocean, architecture, etc.
WeatherFX
If a theme supports it, setting this to "On" will include a backdrop on Glance that reflects the current weather condition.
City + Temp
"On" will include the temperature next to the city name.
City Spoof
"On" will allow you to mask your location's real name with whatever you'd like. Set it below, for City.

"Off" will show the real name of your location.
City
If City Spoof is set to "On" this is where you enter the name you'd like to mask as your location.
Radials
Some complications include a radial border around them reflecting the percent used in the complication.

Set this to "Off" to hide them.
Hour Format
"12" or "24"
Battery Warning
Set the threshold % at which the battery complication radial color will change to the warning color.
UV Index Warning
Set the index # at which the UV Index complication radial color will change to the warning color.
(1-11)
